# Key Trace

## Folder Key (to unlock the name)

Based on the `workspaceKey`

```
{
  workspaceKeyId: string;
  trace: Trace
}
```

Trace:

- rootFolder
- parentFolders
- currentFolder

## Snapshot Key

Based on the `workspaceKey`

Trace:

- rootFolder
- parentFolders
- snapshot

## Document name

Based on the `snapshotKey`

Trace:

- documentName
